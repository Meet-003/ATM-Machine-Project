# Atm machine Project

This is a simple **ATM MACHINE PROJECT** implemented in Python.
It simulates the basic operations of an ATM such as:

- Account balance check
- Deposit
- Withdrawl
- PIN authentication

## Features

- User-friendly console interface
- Secure PIN system
- Tracks transactions

## Technologies used

- Python 3 ---
             |___ Numpy
             |___ Pandas
